Bootstarp

Originally released on Friday, August 19, 2011, we have since had over twenty releases, including two major rewrites with v2 and v3. With Bootstrap 2, we added responsive functionality to the entire framework as an optional stylesheet. Building on that with Bootstrap 3.

Templates

To use Bootstrap templates, you need to follow a few steps. First, you need to set up your environment by running a "watch" or "build" command powered by Gulp, Grunt, or Webpack build tools1. Then, you need to create an HTML page and load Bootstrap via CDN or host it locally.

Blog

The blog is maintained by the Bootstrap team and features articles on topics such as new features, bug fixes, documentation improvements, and more. The blog also includes tutorials, case studies, and other resources to help developers get the most out of Bootstrap.
